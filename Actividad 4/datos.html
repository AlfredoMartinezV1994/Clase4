Explorando las Formas de Mostrar Datos en JavaScript

1. innerHTML

Modifica el contenido de un elemento HTML

document.getElementById("demo").innerHTML = "Hola Mundo";

2. document.write()

Escribe directamente en el documento HTML

document.write('<p>Texto añadido dinámicamente</p>');

Nota: Sobreescribe el documento si se usa después de cargar la página

3. window.alert()

Muestra un cuadro de diálogo modal

alert("¡Esto es una alerta!");

4. console.log()

Escribe en la consola del navegador

console.log("Mira la consola del navegador");

Variables en JavaScript

Declaración y uso de variables

let x = 5;
let y = 6;
let z = x + y;
console.log(z); // 11

Punto y coma opcional

JavaScript tiene Inserción Automática de Punto y Coma (ASI)

let a = 1
let b = 2
[a, b] = [b, a] // Intercambio de valores
console.log(a, b) // 2 1
